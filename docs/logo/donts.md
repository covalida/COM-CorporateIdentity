# Misuse

The logo is a fixed asset, not a starting point. Every rule below protects the same thing: instant, trustworthy recognition. When the mark is recolored, stretched or dressed up with effects, it stops reading as COVALIDA and starts reading as a knock-off — the opposite of what a compliance brand can afford.

::: danger One principle covers most cases
Place the supplied file, unmodified, on a calm background at a legal size. If you're adjusting anything about the logo itself, you've gone too far.
:::

The previews below simulate each mistake so it's easy to recognize. The correct mark is always on the left.

## More don'ts, in one line each

- **Don't crop the symbol or the wordmark** — the mark is always whole.
- **Don't box it in** an unapproved container, border or "badge" ring.
- **Don't set it below minimum size** — see [Construction &amp; Clear Space](/logo/construction).
- **Don't crowd it** — always honor at least 0.5x clear space.
- **Don't screenshot or re-export** a low-quality copy — always start from the master [Downloads](/downloads/).
- **Don't build a print or physical version** — COVALIDA is a 100% digital brand; there is no CMYK, Pantone or merch treatment to invent.

## If in doubt

Use the horizontal logo, unmodified, from [Downloads](/downloads/), on a quiet light or mint background — or the dark variant on dark. Review the full family on [Logo Variants](/logo/variants) and the spacing rules on [Construction &amp; Clear Space](/logo/construction).
